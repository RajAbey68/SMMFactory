## What It Does

A 7-question intake wizard that generates a governed collaboration environment config for professional services client engagements. Outputs:

| Artifact | Description |
|----------|-------------|
| **Cowork YAML Skill** | Ready-to-deploy governance config (channels, folders, HITL register, compliance frame) |
| **Microsoft Graph API Script** | Python script to auto-provision Teams + SharePoint + HITL Register |
| **NotebookLM Setup Script** | `bash` script to create a governed NotebookLM notebook with briefing doc + study guide |

## Usage

### Run the Wizard
```bash
open tools/asimov-ai/index.html
```
Complete the 7-question intake → click **Generate Configuration**.

### Export to NotebookLM
1. Complete wizard → click **Send to NotebookLM** → **Download .sh**  
   *(or use the pre-built `asimov_nlm_setup.sh` and edit the variables at the top)*
2. Authenticate: `nlm login`
3. Run: `bash tools/asimov-ai/asimov_nlm_setup.sh`

### Export PDF
Complete wizard → click **Export PDF** (uses browser print).
